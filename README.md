# HaasTool

HaasTool is an open-source multi-tool platform for Voron printers.

At its core is HOD (Haas One Drive), a shared-drive architecture where a single extruder motor powers multiple independent toolheads.

## Goals

- Low-cost toolchanger
- Minimal filament waste
- Independent heated toolheads
- Open hardware
- Voron compatible
- Community driven

Originally developed for Articulóides production.
